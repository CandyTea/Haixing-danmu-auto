# 自动弹幕工具

> 开源作者：FF。这个仓库中的代码归项目作者所有，并以仓库根目录中的 `LICENSE` 条款对外发布。

一个基于 `tkinter` 的桌面工具，用于手动填写直播间参数并发送单条弹幕或长文本分段弹幕。

## 功能特性

- 支持填写直播间网址并自动解析 `room_id`
- 支持单条弹幕发送
- 支持长文本按固定字数分段发送
- 支持分段循环发送
- 支持不可见字符混淆和随机数字后缀
- 支持查看完整请求头、请求体和返回内容
- 本地保存界面配置到 `config.json`

## 环境要求

- Python 3.11+
- Windows 优先，其他支持 `tkinter` 的 Python 环境理论可运行

## 安装

建议先创建虚拟环境：

```bash
py -3 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 运行

```bash
py -3 main.py
```

## 依赖

项目运行依赖：

- `requests`

## 配置说明

程序启动后可在界面中填写：

- `Authorization`
- `Cookie`
- `Sign`
- `设备 ID`
- `直播间 ID`
- `聊天室 ID`
- `用户 ID`
- `昵称`

这些敏感配置**不会随仓库提交**，默认保存在本地 `config.json` 中，并已被 `.gitignore` 忽略。

如果你想给其他人演示配置格式，可以参考仓库中的 `example_config.json`。

## 发送模式

### 单条弹幕

直接填写单条消息并发送。

### 长文本分段

填写长文本后，设置“每段字数”，程序会自动切分后逐段发送。

例如设置为 `5` 时：

```text
今天天气真不错大家一起冲
```

会被切成：

```text
今天天气真
不错大家一
起冲
```
